📖 Náhled & tisk

Otevře knihu jako elegantní webovou stránku. Přes tisk prohlížeče lze uložit do PDF.

🌐 HTML soubor

Samostatná webová stránka s obálkou, obsahem a celým textem. Otevře se kdekoli.

📚 EPUB

E-kniha pro čtečky (Kindle přes převod, PocketBook, mobily). Včetně obrázků i vlastních fontů.

📄 ODT

Textový dokument pro LibreOffice Writer či Word — k finální sazbě a korekturám.

🟦 DOCX (Word)

Formát Microsoft Word s formátováním textu. Obrázky zachová export do ODT.

🔗 Živá záloha do souboru

Nepropojen — data jsou jen v prohlížeči.

📝 Markdown / text

Prostý text pro další zpracování — sazbu, blog, korektury.

💾 Záloha JSON

Kompletní data pro přenos či zálohu. Import přijme i zálohy z aplikace Tvůrce knih.
