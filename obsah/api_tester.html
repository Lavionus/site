🛰️ API / webhook tester

Jednoduchý HTTP klient pro testování vlastních API a webhooků přímo v prohlížeči.

Odpověď

Zatím žádný požadavek.

  

  
Požadavky se odesílají přímo z tvého prohlížeče (žádný proxy server). Pokud cílové API nemá povolené CORS, prohlížeč odpověď zablokuje — to je omezení bezpečnosti prohlížeče, ne chyba této appky. Pro testování vlastního backendu s CORS to funguje bez problémů.
